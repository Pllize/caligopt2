대시보드
모든 현황을 한눈에 파악

앱을 열면 가장 먼저 보이는 화면입니다. 구매 현황 전체를 빠르게 요약해서 보여줍니다.

①
구매비 / 배송 / 미완료 — 상단 3칸 통계
💰 구매비(확정): 구매대기 제외 실제 결제한 금액 합계. 구매대기 금액이 있으면 주황색으로 추가 표시.
✅ 배송완료 / 🚚 진행: 배송이 끝난 건수 / 배송중·대기 건수.
📋 미구매 · 대기: 아직 구매 기록이 없는 판매처 수 / 구매대기 상태 건수.
②
구매현황 바 — 상태별 클릭으로 상세 조회
미구매 · 구매대기 · 구매완료 · 배송대기 · 배송중 · 배송완료 뱃지를 클릭하면 해당 상태의 목록 팝업이 뜹니다.
미구매 구매대기 구매완료 배송대기 배송중 배송완료
③
판매처별 지출 — 탭해서 상세 펼치기
판매처 카드를 탭하면 해당 판매처에서 구매한 항목 목록이 아래로 펼쳐집니다. 플랫폼 배지와 국내/해외 구분이 함께 표시됩니다.
🇰🇷 국내 🌏 해외 🏷 훗타운 🎵 음총
④
구매 마감일 — D-day 한눈에
마감이 가까울수록 빨간색으로 표시됩니다. 이미 구매한 판매처는 ✓ 구매완료 뱃지가 함께 표시되고, 탭하면 해당 판매처 카드로 스크롤됩니다.
우측 상단의 만료 숨김 버튼으로 기간이 지난 항목을 숨길 수 있어요.
⑤
앨범별 수량 — 탭으로 어디서 샀는지 조회
PHOTOBOOK · POCAALBUM · ID PASS · INVENTORY 각각의 총 수량이 표시됩니다. 수량을 탭하면 그 앨범을 어느 판매처에서 몇 장 구매했는지 팝업으로 확인할 수 있어요.
🛒
구매기록 추가 & 관리
국내/해외 구매를 정확하게 기록하는 법

상단의 + 추가 버튼을 눌러 새 구매 기록을 입력합니다. 국가 → 판매처 순서로 선택하면 해당 판매처에 맞는 입력 폼이 자동으로 나타납니다.

1
🌍 국가 선택
🇰🇷 한국 · 🇯🇵 일본 · 🇨🇳 중국 · 🇹🇼 대만 · 🇺🇸 미국 버튼 중 하나를 탭합니다. 국내 먼저, 해외 뒤로 정렬되어 있어요.
2
🏪 판매처 선택
선택한 국가의 판매처만 가나다순으로 버튼으로 표시됩니다. 판매처를 탭하면 🏷 플랫폼 정보가 자동으로 아래에 표시됩니다.
3
💿 앨범 수량 입력
− / + 버튼 또는 직접 숫자를 입력해 수량을 설정합니다. 수량 0 = 미구매.
⭐ 미공포표시가 된 앨범이 해당 판매처의 미공포 특전 지급 버전입니다.
4
🌏 해외 구매 비용 입력 (해외 판매처만 표시)
직접 결제금액(현지통화): 비워두면 수량 × 단가로 자동 계산됩니다.
⚡ 실제 원화: 입력하면 환율 계산을 무시하고 이 금액이 최우선 적용됩니다.
해외내배송비 · 배송대행비 · 관세 · 국내배송비를 추가 입력하면 예상 합계가 실시간으로 계산됩니다.
5
👤 대리구매 입력 (선택)
사다줄 경우 → 받는 사람: 내가 누군가를 위해 구매할 때 받는 사람 이름 입력.
대리구매 받은 경우 → 사준 사람: 다른 사람이 나 대신 구매해 준 경우.
⚠️
같은 판매처에서 여러 번 구매했다면? 동일 판매처라도 구매 건을 별도로 추가하면 됩니다. 대시보드에서 판매처별 합산 금액이 자동으로 집계됩니다.

배송 상태 업데이트 — 구매 기록 카드 우측 ✏️ 버튼을 눌러 수정 화면에서 상태를 변경합니다.

상태의미
구매대기구매는 하지 않았지만 기록만 해둔 상태 (예: 카트에 담아둠)
구매완료결제 완료, 배송 전
배송대기판매처에서 아직 발송 전
배송중운송장 발급 후 이동 중
배송완료수령 완료

구매기록 정렬 & 필터

정렬 옵션
날짜 ↓ 판매처 금액 상태

같은 정렬 기준을 다시 탭하면 오름차순 ↑ / 내림차순 ↓ 이 전환됩니다. 드롭다운으로 특정 판매처만 필터할 수도 있어요.

캘린더
이벤트 · 마감일 · 구매일 한 화면에서

월간 달력 형태로 판매처 마감일, 오프라인 이벤트, 내 구매 날짜를 함께 볼 수 있습니다.

⏰ 빨간 점
구매 마감일
해당 날짜가 판매 종료일인 판매처. 클릭하면 구매 링크, 현재 구매 상태, 즉시 추가 버튼이 표시됩니다.
📍 보라 점
오프라인 이벤트
팝업스토어 운영 기간. 클릭 시 장소 · 운영시간 · 럭키드로우 여부 등 상세 정보가 표시됩니다.
✓ 파란 점
내 구매 날짜
구매기록에 날짜를 입력한 경우 달력에 표시됩니다. 클릭하면 해당 날의 구매 내역을 확인하고 바로 수정할 수 있어요.
🎵 보라 띠
고정 발매일
4월 13일 DIGITAL RELEASE, 4월 14일 ALBUM RELEASE — 항상 표시됩니다.
🔀
온라인 / 오프라인 토글
캘린더 상단의 🛍 온라인판매 📍 오프라인 버튼으로 각각의 이벤트를 ON/OFF 할 수 있습니다.

날짜 셀을 탭하면 그 날의 이벤트 목록이 달력 아래에 펼쳐집니다. 오프라인 이벤트는 장소 · 운영시간 · 주소 · 럭키드로우 여부가 표시되고, ✏️ 정보수정 버튼으로 바로 편집 창을 열 수 있어요.

[음총] 표기 판매처란?
🎵 음총 배지는 음원총판을 통해 판매되는 판매처를 의미합니다. 공식 판매처와 다른 유통 경로로, 가격이 다를 수 있어요.

환율 & 단가
해외 구매 비용 자동 환산

헤더의 💱 아이콘을 탭하면 현재 적용 중인 환율이 팝업으로 표시됩니다. ↻ 갱신 버튼으로 실시간 환율을 가져올 수 있어요.

💡
실제 결제 원화를 직접 입력하는 경우
해외 구매 기록에서 ⚡ 실제 원화 필드에 값을 입력하면 환율 계산을 무시하고 해당 금액을 기준으로 합계가 산정됩니다. 카드사 환율이나 수수료가 포함된 실제 청구 금액을 입력할 때 유용해요.

환율은 앱을 열 때 1시간 이상 지났으면 자동으로 갱신을 시도합니다. 인터넷이 없는 환경이거나 API가 응답하지 않는 경우 관리 탭의 환율 설정에서 수동으로 체크하고 직접 입력할 수 있습니다.

💾
데이터 백업 & 복원
폰 ↔ 컴퓨터 데이터 이동 방법

구매 기록은 각자의 브라우저 저장소(localStorage)에 자동으로 저장됩니다. 별도 서버가 없어서 개인정보가 외부에 전송되지 않아요. 단, 브라우저 데이터를 지우면 기록도 사라지므로 정기적으로 백업을 권장합니다.

⚠️
브라우저 초기화 전 반드시 내보내기!
브라우저 캐시/데이터 삭제, 시크릿 모드 사용, 브라우저 재설치 시 구매 기록이 삭제됩니다. 중요한 기록은 꼭 JSON 파일로 내보내 두세요.
⬇
내보내기 — 파일로 저장
헤더의 💾 → ⬇ 내보내기 버튼을 누르면 구매 기록 전체가 PLAVE_Caligo2_날짜.json 파일로 다운로드됩니다.
⬆
불러오기 — 파일에서 복원
💾 → ⬆ 불러오기에서 JSON 파일을 선택하면 구매 기록이 복원됩니다.
현재 기록이 덮어씌워지므로 불러오기 전에 먼저 내보내기로 백업을 해두세요.

📱 폰 ↔ 💻 컴퓨터 이동 방법

1
폰에서 내보내기
폰 브라우저에서 💾 내보내기 → JSON 파일이 다운로드 폴더에 저장됩니다.
2
파일 전송
카카오톡 나에게 보내기, 에어드롭, 이메일 등 편한 방법으로 파일을 컴퓨터로 전송합니다.
3
컴퓨터에서 불러오기
컴퓨터 브라우저에서 앱을 열고 💾 불러오기로 JSON 파일을 선택하면 완료!
✨
팁 & 자주 묻는 질문
더 편하게 쓰는 방법
📌
홈 화면에 추가하면 앱처럼 사용 가능!
모바일 브라우저에서 공유 → 홈 화면에 추가를 선택하면 앱 아이콘이 생성되어 일반 앱처럼 전체 화면으로 사용할 수 있습니다.
궁금한 점답변
구매기록이 사라졌어요 브라우저 캐시가 초기화된 것입니다. 이전에 내보내기로 저장한 JSON 파일이 있다면 불러오기로 복원하세요.
판매처를 추가하고 싶어요 업데이트 최대한 빨리 해보겠습니다.
같은 판매처에서 여러 종류를 샀어요 하나의 구매 기록에서 여러 앨범 수량을 함께 입력할 수 있습니다. 수량이 0인 앨범은 저장 시 제외됩니다.
누군가 대신 구매해줬어요 구매기록 입력 시 "대리구매 받은 경우 — 사준 사람" 필드에 이름을 입력하세요.
내가 누군가를 위해 사줬어요 "사다줄 경우 — 받는 사람" 필드에 이름 입력. 대시보드의 "사다준 목록"에서 집계됩니다.
다크/라이트 모드 바꾸려면? 헤더 우측의 🌙 / ☀️ 아이콘을 탭하세요. 설정이 자동으로 저장됩니다.
미공포 특전이란? 판매처마다 ⭐ 미공포 표시된 앨범 버전이 특전 지급 대상입니다. 구매기록 입력 시 해당 앨범에 강조 표시됩니다.
