💡 사용법
  1. 위에서 GitHub 설정을 입력하고 연결 테스트를 확인하세요
  2. 미공포 탭에서 카드를 클릭해 이미지를 업로드하세요
  3. 이미지가 자동으로 GitHub images/ 폴더에 push됩니다
  4. unpo.js도 자동으로 업데이트되어 함께 커밋됩니다
  5. 약 1분 후 GitHub Pages에 반영됩니다

📁 저장 방식 안내
🚀 GitHub에 저장 버튼 클릭 시:

  1. 업로드된 이미지 → images/ 폴더에 push

       └── var UNPO_DATA = {...}        ← 미공포 세트·이미지

  약 1분 후 GitHub Pages에 자동 반영됩니다.
